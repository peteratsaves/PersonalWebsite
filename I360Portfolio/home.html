Project Work

Light Sensor RobotGitHubReport

Robotics Class: October 2021

We had to build a robot which can be placed anywhere in a room and find a light source. This included attaching a light sensor that scans for light and having the robot move after periodically scanning for the strongest direction of the light. When no light is found, we implemented a case to have the robot move forward and turn until it finds a light source that meets the threshold.

Maze Traversal RobotGitHubReport

Robotics Class: November 2021

Our task was to build a robot which starts at the opening of a maze and can travel to the end of it without being moved or manually controlled throughout the maze. Our approach was to have the robot track the wall and follow the wall until it gets to the end of the maze. This required the use of ultrasonic sensor which scanned for open space, as well as bump sensors that were used to know when the robot was against or away from the wall. We finished in 2nd place in our class competition.

Manipulation RobotGitHubReport

Robotics Class: December 2021

Our assignment was to build a 2-joint robot which can pick up 3 cans in a row, and move them behind the robot. After building our robot, we gathered data points to determine how the arms on the robot should move in order to pick up the cans. We used linear regression to generate an equation for each joint relative to the distance (ultrasonic sensor value), and we were able to pick up all 3 cans.
